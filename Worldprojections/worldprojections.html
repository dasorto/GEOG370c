In this project I learned how to display images in different projections

Alex Sorto

In order to display the different projections in QGIS, I first imported the vector layer from Natural Earth of all countries provided. Then I would go to the bottom right corner in QGIS and press the button besides messages. There you can change the different projections and click okay to the one you want searching by code or name.

WGS84 Projection

We can see that the distorted areas are closer to the top and bottom. The map is stretched in those areas.

Aitoff Projection

This projection shows how the map is wrapped or folded in the top and bottom. Thus, the circles move in position.

Pseudo-Mercator Projection

The Pseudo-Mercator Projection shows how countries in the top and bottom are stretched or shown bigger. For example, we can see that Greenland is about the size of Africa.

Sphere Winkle Projection

This projection is interesting since it folds almost perfectly like a rhombus. Thus, the circles are bigger in the top and bottom.

World Cylindrical Equal Area Projection

This projection shows how the edges in the top and bottom are crushed or made smaller.

World Equidistant Conic Projection

In this projection we can see from a view from a top the globe, and how the farther from the center the countries are smaller and more stretched.

North Pole Azimuthal Projection

This projection looks like the last projection but this one has a different view from a top and is even more stretched farther from the center.

Sphere Eckert 2 Projection

This projection is similar to the Sphere Winkle Projection but its shape is more like a rhombus or trapezoid.

World Eckert 3 Projection

The World Eckert 3 Projection represents a cylinder and the farther away from the center a coutry is, the more stretched the map is there.
